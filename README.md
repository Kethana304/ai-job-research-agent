# 🤖 AI Job Research Agent

An autonomous multi-step AI agent that researches job market skills 
and generates personalized learning roadmaps for AI Engineering roles.

## Tech Stack
- Python
- Groq API (LLaMA 3.3 70B)
- Agentic AI — multi-step reasoning & task decomposition

## How it Works
1. Agent researches top AI Engineer skills
2. Analyzes and prioritizes skills for freshers
3. Generates a 4-week learning roadmap with free resources

## Run it
pip install groq python-dotenv
python job_agent.py
